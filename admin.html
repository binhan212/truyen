Thêm bộ truyện mới

Tên thư mục trong assets/files/ chứa các file .txt. Ví dụ: caudao hoặc truongsinhtienmon

Truyện đã thêm (localStorage)

    Để truyện hiển thị chương:
    1. Tạo thư mục assets/files/slug/
    2. Đặt các file .txt vào thư mục đó
    3. Chạy .\assets\files\update-manifest.ps1 để cập nhật danh sách chương

    Để lưu vĩnh viễn: Sao chép JSON bên dưới vào file assets/data/novels.json
